# CYBERPUNK ISO MMO -- PIXEL ART PROMPT LIBRARY

## Project Identity

-   2D Isometric
-   Pixel Art Only
-   Cyberpunk 2001
-   Dark Diablo-ish Mood
-   MMO Readability Priority
-   96px Tall Characters
-   64×32 Isometric Ground Tiles (2:1 ratio)

All assets must remain readable in multiplayer density.

# PIXEL DISCIPLINE RULES (CRITICAL)

-   No anti-aliasing\
-   No subpixel blur\
-   Clean pixel clusters\
-   Limited palette per asset\
-   High contrast edges\
-   No painterly gradients\
-   No photorealism\
-   No 3D rendering look\
-   Designed for MMO readability

------------------------------------------------------------------------

# GENERATION ORDER (IMPORTANT)

Do NOT start with characters.

Generate in this order:

1.  Ground tiles (asphalt, sidewalk, variants)\
2.  Street props (lamp, dumpster, pole)\
3.  One modular building set\
4.  Neon sign set\
5.  THEN characters

World first. Player later.
